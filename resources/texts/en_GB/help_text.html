Chrona – Help

What the application does

Chrona is used to move or copy files and folders to another location. You can choose what to transfer, where to place it, and how files should be processed during the transfer.

File transfer

  • Drag & Drop area – drag files or folders here
  • Select files / folders – opens a dialog for manual selection
  • Select destination folder – defines where files will be moved or copied

After selection, the application displays:

  • number of selected items
  • number of files
  • number of folders

Additional options:

  • Show list – displays selected items, individual entries can be removed
  • Clear list – removes all selected items

Click Start to begin the transfer using current settings.

Transfer settings

Default paths

You can set default source and destination folders at the top. These paths are used when opening dialogs in the main window, so you don’t need to navigate through the entire disk every time.

Folder structure

  • Create folders based on date (year / month / day)
  • Month can only be enabled if year is enabled
  • Day can only be enabled if month is enabled
  • Option to create subfolders by file type
  • Include hidden files

File filters

  • Enable main filter
  • Select categories (documents, text, Office, images, music, archives)
  • Option to define custom extensions (e.g. .pdf)

File naming

  • Default name or custom name
  • When using a custom name, the following options are available:
    • timestamp
    • counter
  • At least one of these options must be enabled

Additional options

  • Move (delete source files) or show failed files after completion

Progress and results

  • Progress is displayed during transfer (e.g. 1/10)
  • The process can be interrupted and confirmed
  • After completion, the result is displayed
  • Failed files can be viewed in a list

Notes

  • When selecting a folder, all subfolders are processed automatically
  • Duplicate files are skipped
